# webots-quadruped-simulation

### Setup 
Um das zuletzt trainierte Model in Webots anzuwenden müssen folgende Steps durchgeführt werden.

1. Download Webots
2. Clone Repository
3. Install requirements.txt
4. Open Webots
5. File > Open World > Select > 25-12-quadruped_spot.wbt
6. Press Simulation Play Button
7. Click in Simulation-Window
8. UpArrow > Increase Velocity
9. Down > Arrow Reduce Velocity
10. Space > Set Velocity to 0
