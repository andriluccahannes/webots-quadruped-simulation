# webots-quadruped-simulation

### Setup 
Um das zuletzt trainierte Model in Webots anzuwenden müssen folgende Steps durchgeführt werden.

1. Download Webots
2. Clone Repository
3. Open Webots
4. File > Open World > Select > 25-12-quadruped_spot.wbt
5. Press Simulation Play Button
6. Click in Simulation-Window
7. UpArrow > Increase Velocity
8. Down > Arrow Reduce Velocity
9. Space > Set Velocity to 0
